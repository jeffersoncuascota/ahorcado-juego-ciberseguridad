# El Impacto de las Nuevas Tecnologías en la Sociedad: Visualización del Futuro

## Datos del Grupo
- **Nombre del Proyecto**: El impacto de las nuevas tecnologías en la sociedad: visualización del futuro
- **Integrantes**:
  - Jefferson Cuascota
- **Asignatura**: LOGICA DE PROGRAMACION
- **Fecha de Entrega**: 20 de Abril de 2026 (Semana 8)

## Objetivo del Programa
Desarrollar un software interactivo que permita **visualizar el impacto de las nuevas tecnologías** en diferentes áreas de la sociedad y reflexionar sobre posibles escenarios futuros (positivo, neutro y negativo).

El proyecto incluye un **juego educativo del Ahorcado** temático sobre tecnologías emergentes, que sirve como herramienta lúdica para reforzar conceptos y fomentar la reflexión sobre el rol de la tecnología en la sociedad.

## Principales Funcionalidades

### 1. Juego del Ahorcado Educativo
- Más de 20 palabras relacionadas con programación, IA, ciberseguridad y tecnologías emergentes.
- Dibujo del ahorcado con 6 etapas (usando **tuplas** para almacenar las etapas).
- Sistema de intentos, letras usadas y validaciones.
- Mensajes motivadores y reflexión final sobre el impacto tecnológico.
- Opción de jugar múltiples veces.

### 2. Integración de las 4 Unidades de la Materia
- **Unidad 1 y 2**: Programación Orientada a Objetos (clases, funciones) y Estructuras de Datos (listas, conjuntos, tuplas).
- **Unidad 3**: Algoritmos (lógica de validación, verificación de victoria/derrota, manejo de flujo).
- **Unidad 4**: **Programación Funcional** (uso de funciones puras, comprensión de listas, type hints y funciones de orden superior).

## Tecnologías y Herramientas Utilizadas
- Lenguaje: **Python 3**
- Estructuras: Listas, Conjuntos (`set`), **Tuplas**
- Paradigmas: Programación Orientada a Objetos + **Programación Funcional**
- Control de versiones: **GitHub**
